About the system


1) Admin/user 

There are 2 types of users in the system. Admins and users. They have different rights.
      Admin:
 For the admin, data and visitor schedules of all users are available.
 Admin can add, edit and remove users/events.
      User:
 For user are avaliable: own graphic, all users data and schedule of lessons and events. 
 
 2) Face-recognition system
 
 The app includes face-recognition system. It is training on data of images (photo of users). Then at startup of the app the system
 recognized the user and send data (time, name of user) to the database. On the profile user can see the visit schedule.
